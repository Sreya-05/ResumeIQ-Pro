# ResumeIQ Pro

## Objective
ResumeIQ Pro is an AI-powered resume builder that helps users create professional resumes, analyze ATS compatibility, generate AI-assisted content, and export resumes as PDF.

## Features
- Multi-step Resume Builder
- ATS Analyzer
- AI Assistant
- Resume Templates
- PDF Export
- Draft Saving
- Local Storage Persistence
- Responsive UI
- Dark Mode

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Zustand
- React Hook Form

## Installation

npm install

npm run dev

## Build

npm run build
