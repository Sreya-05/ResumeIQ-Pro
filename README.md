# ResumeIQ Pro – AI-Powered Resume Builder & ATS Optimizer

## Overview

ResumeIQ Pro is a modern AI-powered resume building platform designed to help students, graduates, and professionals create industry-standard resumes with ease. The application combines intelligent resume creation, ATS compatibility analysis, AI-assisted content generation, and PDF export capabilities within a responsive and user-friendly interface.

The platform enables users to build professional resumes, receive AI-generated suggestions, analyze resume performance against job descriptions, and optimize content for applicant tracking systems (ATS).

---

## Problem Statement

Many job seekers struggle to create professional resumes that satisfy modern recruitment standards and ATS requirements. Traditional resume creation methods often result in poor formatting, weak content, and low ATS scores.

ResumeIQ Pro addresses these challenges by providing an intuitive resume-building experience enhanced with AI-driven recommendations and ATS optimization features.

## Project Architecture

The application follows a component-based architecture using React and TypeScript. State management is handled through Zustand, while form validation and handling are managed using React Hook Form. Mock backend services simulate real-world API interactions, making the application fully functional without requiring a backend server.

## Installation & Setup

### Clone Repository

```bash
git clone <repository-url>
cd resumeiq-pro
```

### Install Dependencies

```bash

```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
